# WeChatDice

Generate the point you want using WeChat's dice emoticon.

## Description

适配 iOS 9

可以任意指定微信中的骰子表情和剪刀石头布表情的结果。

在「设置」中进行配置。

![](screenshot.PNG)

## Install

在 Cydia™ 中添加源 [repo.wangjinli.com](http://repo.wangjinli.com) 搜索安装。
